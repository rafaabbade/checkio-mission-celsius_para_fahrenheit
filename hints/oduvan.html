Como converter Celsius para Fahrenheit?

Utilize a fórmula fahrenheit = celsius * 9/5 + 32 para converter a temperatura de Celsius para Fahrenheit.

Como fazer a saída no formato certo?

Para retornar a eficiência do combustível e a recomendação, use uma tupla: return (eficiencia, recomendacao)

Minha fórmula está certa, mas o número de casas decimais não.

Use a função round(valor, 1) para garantir que a temperatura tenha uma casa decimal.
